# github-trends
Analyzing trends on Github using topic models and machine learning.

## Raw URLs

The Github repository URLs can be found in the _raw urls_ directory, within 7z files. Each archive contains 1 million repository URLs, sorted from oldest to newest. For example, _1m.7z_ contains the first 1 million repository URLs. Within the 7z archives, there are files each containing 50,000 URLs. For example, within the _1m.7z_ archive, the file _0.txt_ contains the first 50,000 links, the file _50,000.txt_ contains the next 50,000 urls, etc. _2m.7_ contains the next 1 million URLs, contained within _1000000.txt_, _1050000.txt_, etc.
